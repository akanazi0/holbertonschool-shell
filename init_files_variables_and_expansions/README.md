# Shell, Init Files, Variables and Expansions

This repository contains Bash scripts written for the **init_files_variables_and_expansions** the fouth project at Holberton School Shell Projects.

## Files Description

* **0-alias** - Creates an alias.
* **1-hello_you** - Prints hello user, where user is the current Linux user.
* **2-path** - Adds a directory to the PATH.
* **3-paths** - Counts the number of directories in the PATH.
* **4-global_variables** - Lists environment variables.
* **5-local_variables** - Lists all local variables, environment variables, and functions.
* **6-create_local_variable** - Creates a new local variable.
* **7-create_global_variable** - Creates a new global variable.
* **8-true_knowledge** - Prints the result of an arithmetic addition.
* **9-divide_and_rule** - Prints the result of a division.
* **10-love_exponent_breath** - Prints the result of an exponent calculation.
* **11-binary_to_decimal** - Converts a number from binary to decimal.
* **12-combinations** - Prints all combinations of two letters.
* **13-print_float** - Prints a number with two decimal places.
* **14-decimal_to_hexadecimal** - Converts a number from decimal to hexadecimal.
